# nano-dotfiles
Dotfiles for i3wm/polybar

## Dependencies
### Main
- i3
- polybar
- alacritty
- rofi

### Optional
- helix
- fish
- maim
- xclip

## Setup
```bash
git clone https://github.com/Ad4ndi/pico-dotfiles
cp -r pico-dotfiles/.config/* ~/.config/
rm -rf pico-dotfiles
```
